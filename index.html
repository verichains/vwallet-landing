TeamWallet
A wallet that holds money for the team. All transactions must be approved by a required number of team members. The wallet also supports adding or removing members, as well as changing the number of approval required per transaction.
Metamask Integration
The VWallet app works like a hardware wallet. All transactions, contract calls, dApp integrations,... via Metamask on the web browser will be sent to user's device, and must be approved and signed by user (via VWallet app) in order to be executed. Don't need to import private keys directly to Metamask!
